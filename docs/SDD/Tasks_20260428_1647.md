**Alvo:** `src/cartao_mercadopago.py` e `src/cartao_santander.py`

**Ação:**
1. Criar dicionário de envelope com os campos obrigatórios abaixo.

3. Transporte zero-disk: `print(json.dumps(envelope))` exclusivamente para `stdout`.

**Formato do envelope:**
```json
{
  "id_lote": "{EMISSOR}-{YYYYMMDD}-{HHMMSS}",
  "data_processamento": "2026-04-28T14:30:22",
  "emissor": "santander|mercadopago",

  "avisos": [],

```

> ⚠️ `id_lote` formato `{EMISSOR}-{YYYYMMDD}-{HHMMSS}` — UUID4 **não usar**.

**Critério de aceite:**
- Campo `id_lote` presente no envelope e em cada lançamento.
- Campo `avisos` sempre presente (lista vazia em execução limpa).
- Zero escrita em disco durante execução.

## TASK-05: Integração VBA Exec e Fail-Fast (Excel) [BR-03]

**Alvo:** `Extratores.xlsm` (Módulo `ModComum`)

**Ação:**
1. Substituir `.Run` por `oExec = WScript.Shell.Exec(...)`.
2. Capturar buffers nesta ordem (evita deadlock de pipe):
   ```vba
   oExec.StdIn.Close             ' fechar stdin primeiro
   jsonStr = oExec.StdOut.ReadAll
   errStr  = oExec.StdErr.ReadAll
   ```
3. `oExec.ExitCode <> 0` → `MsgBox errStr`, `Exit Sub`.
4. `ExitCode = 0` com `errStr` não-vazio → `MsgBox` de aviso, não abortar.

**Critério de aceite:**
- `C:\Temp\extratores_output.json` nunca criado.
- Falha no Python → Excel não grava nada.

> ⚠️ Senha entregue via `StdIn.WriteLine` antes do `StdIn.Close` — não via `--password`.

## TASK-09: Entrega Segura de Senha + Ajustes Docker (Python + VBA)

**Alvo:** `src/pdf_decrypt.py`, `src/cartao_mercadopago.py`, `src/cartao_santander.py`, `vba/ModComum.bas`

**Status:** ✅ CONCLUÍDA — todos os arquivos atualizados em 2026-04-28.

**Contexto:** Três problemas resolvidos em conjunto por tocarem os mesmos arquivos:
1. Senha trafegava via `--password` CLI — visível em `ps aux` e logs de processo
2. `sys.stdout = io.TextIOWrapper(...)` era workaround Windows pesado, incompatível com Docker
3. `pdf_decrypt.py` sem tratamento de erro — senha incorreta gerava traceback em vez de mensagem limpa

**Ações executadas:**

**`src/pdf_decrypt.py`**
- Adicionado `try/except` em `descriptografar()` — erro de senha gera mensagem limpa no `stderr` + `sys.exit(1)` em vez de traceback

**`src/cartao_mercadopago.py` e `src/cartao_santander.py`**
- `sys.stdout = io.TextIOWrapper(sys.stdout.buffer, encoding="utf-8")` → `sys.stdout.reconfigure(encoding="utf-8")`
- `import io` removido de `cartao_mercadopago.py` (não tinha outros usos)
- Leitura de senha via stdin adicionada após `parse_args()`:
  ```python
  password = args.password
  if not sys.stdin.isatty():
      linha = sys.stdin.readline().strip()
      if linha:
          password = linha
  ```
- `processar_pasta(input_path, args.password)` → `processar_pasta(input_path, password)`

**`vba/ModComum.bas`**
- Removido bloco `--password` do comando CLI
- Adicionado envio via stdin antes do `ReadAll`:
  ```vba
  If Len(Trim(senha)) > 0 Then
      oExec.StdIn.WriteLine senha
  End If
  oExec.StdIn.Close
  ```

**Critério de aceite (verificar ao montar o xlsm):**
- Senha não aparece no comando CLI nem em `ps aux`
- Senha incorreta (Santander) → `MsgBox` com mensagem legível, nenhum dado gravado
- Mercado Pago (sem senha) → stdin fechado imediatamente, processamento normal
- Ambos os extratores funcionam em Docker sem o workaround `io.TextIOWrapper`

## Decisões de Arquitetura Incorporadas nesta Versão

| ID | Decisão | Rationale |
|----|---------|-----------|
| A1 | `id_lote` formato `{EMISSOR}-{YYYYMMDD}-{HHMMSS}` | Auditável por humano (BR-04) |
| A2 | Modo escrita APPEND + rollback por ID_Lote | Histórico preservado; reversão granular (BR-04) |
| A3 | Regex de parcela por emissor, não genérica | Evita falsos positivos; testada por PDF real |
| A4 | `StdOut.ReadAll` antes de `ExitCode` | Previne deadlock de pipe (TASK-05) |
| A5 | `errStr` não-vazio com ExitCode=0 → aviso, não abort | Warnings técnicos visíveis sem bloquear operação |
| A6 | `sys.exit(2)` para argumento ausente explicitado | Falha rápida e clara para o operador |
| A7 | Descriptografia centralizada em `pdf_decrypt.py` | Evita duplicação; novos emissores com uma importação |
| A8 | ScriptControl substitui parser JSON manual no VBA | VBA como acionador puro (NFR-01); elimina locale |
| A9 | Senha entregue via stdin, não `--password` | Não expõe credencial em `ps aux`; pronto para Docker multi-usuário |
| A10 | `sys.stdout.reconfigure()` em vez de `TextIOWrapper` | API nativa Python 3.7+; cross-platform sem substituir stdout |
