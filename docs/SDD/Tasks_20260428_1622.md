## TASK-05: Integração VBA Exec e Fail-Fast (Excel) [BR-03]

**Alvo:** `Extratores.xlsm` (Módulo `ModComum`)

**Ação:**
1. Substituir `.Run` por `oExec = WScript.Shell.Exec(...)`.
2. Capturar buffers nesta ordem (evita deadlock de pipe):
   ```vba
   jsonStr = oExec.StdOut.ReadAll
   errStr  = oExec.StdErr.ReadAll
   ```
3. `oExec.ExitCode <> 0` → `MsgBox errStr`, `Exit Sub`.
4. `ExitCode = 0` com `errStr` não-vazio → `MsgBox` de aviso, não abortar.

**Critério de aceite:**
- `C:\Temp\extratores_output.json` nunca criado.
- Falha no Python → Excel não grava nada.

> ⚠️ `StdOut.ReadAll` obrigatoriamente **antes** de verificar `ExitCode` — previne deadlock de pipe.

## TASK-07: Módulo Compartilhado de Descriptografia (Python)

**Alvo:** `src/pdf_decrypt.py` (novo), `src/cartao_mercadopago.py`, `src/cartao_santander.py`

**Ação:**
1. Criar `src/pdf_decrypt.py` com `descriptografar(pdf_path, password)`.
2. Atualizar ambos os extratores para importar e usar o módulo — remover qualquer lógica local.

```python
# src/pdf_decrypt.py
import io
from pathlib import Path

def descriptografar(pdf_path: Path, password: str):
    if not password:
        return pdf_path
    import pikepdf
    try:
        with pikepdf.open(pdf_path, password=password) as pdf:
            buf = io.BytesIO()
            pdf.save(buf)
            buf.seek(0)
        return buf
    except Exception as e:
        import sys
        print(f"ERRO: falha ao descriptografar {pdf_path.name}: {e}", file=sys.stderr)
        sys.exit(1)
```

**Critério de aceite:**
- Extratores não contêm lógica de descriptografia própria.
- Zero PDFs descriptografados em disco.
- Novos extratores ganham suporte a senha com uma importação.

## TASK-08: Substituir Parser JSON Manual por ScriptControl (VBA) [NFR-01]

**Alvo:** `vba/ModComum.bas`

**Contexto:** O `ModComum` original continha ~70 linhas de parser JSON manual (`ExtrairCampo`, `ExtrairArray`, loop com `InStr`) — lógica de interpretação de dados que viola NFR-01 (inteligência fora do VBA). Além disso, `CDbl(Replace(fValor, ".", ","))` era um workaround de locale frágil que derivava do parser manual.

**Ação:**
1. Remover as funções `ExtrairCampo` e `ExtrairArray` inteiramente.
2. Substituir o loop de parsing por `MSScriptControl.ScriptControl`:
   ```vba
   Dim sc As Object
   Set sc = CreateObject("MSScriptControl.ScriptControl")
   sc.Language = "JScript"
   sc.ExecuteStatement "var env = " & jsonStr

   total = sc.Eval("env.lancamentos.length")
   For i = 0 To total - 1
       prefix = "env.lancamentos[" & i & "]."
       .Cells(rowNum, 7).Value = CDbl(sc.Eval(prefix & "valor"))  ' número nativo, sem Replace
       .Cells(rowNum, 6).Value = sc.Eval("env.lancamentos[" & i & "].parcela || ''")
       ...
   Next i
   ```
3. Substituir `ExtrairArray("avisos")` por loop via `sc.Eval("env.avisos.length")`.
4. Adicionar handler `ErroParse:` para JSON inválido.

**Status:** ✅ CONCLUÍDA — `vba/ModComum.bas` atualizado em 2026-04-28.

**Resultado:**
- `ExtrairCampo` e `ExtrairArray` removidas.
- `CDbl(Replace(fValor, ".", ","))` → `CDbl(sc.Eval(...))` — sem dependência de locale.
- `parcela null` tratado via JScript ternary `|| ''` — sem lógica condicional extra no VBA.
- JSON malformado → `MsgBox` com diagnóstico + primeiros 200 chars para debug.
- `ModComum.bas` reduzido de ~220 para ~130 linhas.

## Decisões de Arquitetura Incorporadas nesta Versão

| ID | Decisão | Rationale |
|----|---------|-----------|
| A1 | `id_lote` formato `{EMISSOR}-{YYYYMMDD}-{HHMMSS}` | Auditável por humano (BR-04) |
| A2 | Modo escrita APPEND + rollback por ID_Lote | Histórico preservado; reversão granular (BR-04) |
| A3 | Regex de parcela por emissor, não genérica | Evita falsos positivos; testada por PDF real |
| A4 | `StdOut.ReadAll` antes de `ExitCode` | Previne deadlock de pipe (TASK-05) |
| A5 | `errStr` não-vazio com ExitCode=0 → aviso, não abort | Warnings técnicos visíveis sem bloquear operação |
| A6 | `sys.exit(2)` para argumento ausente explicitado | Falha rápida e clara para o operador |
| A7 | Descryptografia extraída para `pdf_decrypt.py` compartilhado | Evita duplicação; novos emissores ganham suporte com uma importação |
| A8 | ScriptControl substitui parser JSON manual no VBA | VBA como acionador puro (NFR-01); elimina dependência de locale |
