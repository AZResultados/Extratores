## TASK-12: Cadastro de Clientes + Botão Unificado de Processamento [NFR-01]

**Status:** ✅ CONCLUÍDA — 2026-04-28

**Contexto:** O cliente era um valor fixo na aba Config do Excel (B4). Com múltiplos clientes, o operador precisaria editar a célula a cada uso. A solução integra um cadastro de clientes ao banco SQLite e apresenta um dropdown numerado ao processar, eliminando toda configuração manual de pasta e cliente.

---

### Novos arquivos Python

**`src/db_cliente.py`**
- Tabela `clientes (nome TEXT PRIMARY KEY, base_dir TEXT)`
- `set_cliente(nome, base_dir)` — upsert
- `get_cliente(nome)` → `{nome, base_dir}` ou `None`
- `listar_clientes()` → `[(nome, base_dir)]`
- `remover_cliente(nome)`
- Compartilha o mesmo `dados.db` de `db_senha.py` (tabelas separadas)

**`src/setup_cliente.py`**
- `add <nome> <base_dir>` — registra/atualiza cliente
- `list` — saída `nome|base_dir` por linha (parseável pelo VBA)
- `remove <nome>` — remove cliente
- `get <nome>` — retorna apenas o `base_dir`

---

### Alteração em `src/extrator.py`

Validação de caminho atualizada — verifica se `--cliente` aparece em **qualquer nível** do caminho (não só na pasta imediata):

```python
# Antes: só verificava pasta imediata
if input_path.name != args.cliente: ...

# Depois: verifica todo o caminho (compatível com subpastas por mês)
partes = [p.name for p in input_path.parents] + [input_path.name]
if args.cliente not in partes:
    avisos.append(...)
```

---

### Novos módulos VBA

**`ModProcessar.bas`** — botão único "Processar"
```
1. SelecionarCliente()  → dropdown numerado do banco
2. SelecionarPasta()    → folder picker iniciando no base_dir do cliente
3. ProcessarExtrator()  → chama extrator.py
```

**`ModClientes.bas`** — gerenciamento de clientes
- `CadastrarCliente()` — InputBox para nome + folder picker para base_dir
- `RemoverCliente()` — dropdown + confirmação

---

### Módulos VBA atualizados

**`ModComum.bas`** — helpers compartilhados adicionados:
- `ObterListaClientes()` — chama `setup_cliente.py list`, retorna array `nome|base_dir`
- `SelecionarCliente(outBaseDir)` — dropdown numerado, retorna nome e base_dir
- `SelecionarPasta(titulo, startPath)` — folder picker com ponto de partida

**`ModSenhas.bas`** — `CadastrarSenhaPDF` e `RemoverSenhaPDF` usam `SelecionarCliente()` em vez de InputBox de texto livre

**`ModMP.bas` / `ModSantander.bas`** — redirecionam para `Processar()` (retrocompatibilidade com botões existentes)
