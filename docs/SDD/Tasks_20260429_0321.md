## TASK-13: ModConfig — Caminhos em Módulo VBA [NFR-01]

**Status:** ✅ CONCLUÍDA — 2026-04-28

**Contexto:** Caminhos de executáveis e scripts estavam na aba Config do Excel (células B1–B4). Isso cria dependência de planilha, dispersa configuração e viola o princípio de manter o Excel limpo. A aba Config pode agora ser removida inteiramente.

**Implementação:**

`vba/ModConfig.bas` (novo):
```vba
Private Const BASE_DIR As String = "C:\Dev\projetos\Extratores"

Public Function PythonExe() As String
    PythonExe = BASE_DIR & "\venv\Scripts\python.exe"
End Function
Public Function ExtratorScript() As String
    ExtratorScript = BASE_DIR & "\src\extrator.py"
End Function
Public Function SetupSenhaScript() As String
    SetupSenhaScript = BASE_DIR & "\src\setup_senha.py"
End Function
Public Function SetupClienteScript() As String
    SetupClienteScript = BASE_DIR & "\src\setup_cliente.py"
End Function
```

**Para mudar de máquina ou pasta:** alterar apenas `BASE_DIR` em `ModConfig.bas`.

## TASK-14: Logging Estruturado — ✅ CONCLUÍDA — 2026-04-29

**Contexto:** Sem observabilidade em produção — falhas silenciosas, impossível auditoria de processamento.

**Implementação:**

`src/logger.py` (novo):
- `RotatingFileHandler` → `~/.extratores/extrator.log` (5 MB, 3 backups)
- Formato: `%(asctime)s | %(levelname)-8s | %(name)s | %(message)s`
- Nível padrão: INFO (override via env `EXTRATORES_LOG_LEVEL`)
- Fallback para stderr se o diretório não puder ser criado
- `_setup()` idempotente — safe para múltiplos imports

**Módulos integrados:**

| Módulo | Logger | Eventos |
|--------|--------|---------|
| `extrator.py` | `extratores.extrator` | início/lote/PDF processado/erro |
| `pdf_router.py` | `extratores.router` | emissor detectado/aviso senha |
| `cartao_santander.py` | `extratores.santander` | início/OK/divergência/titular |
| `cartao_mercadopago.py` | `extratores.mercadopago` | início/OK/divergência/cartão |
| `db_senha.py` | `extratores.db` | CRUD senhas (sem expor o valor) |
| `db_cliente.py` | `extratores.db` | CRUD clientes |

**Restrições:** senhas, CPFs e números completos de cartão nunca são logados. Falha no logging não causa `sys.exit(1)`.
