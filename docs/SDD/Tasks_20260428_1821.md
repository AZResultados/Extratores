## TASK-13: ModConfig — Caminhos em Módulo VBA [NFR-01]

**Status:** ✅ CONCLUÍDA — 2026-04-28

**Contexto:** Caminhos de executáveis e scripts estavam na aba Config do Excel (células B1–B4). Isso cria dependência de planilha, dispersa configuração e viola o princípio de manter o Excel limpo. A aba Config pode agora ser removida inteiramente.

**Implementação:**

`vba/ModConfig.bas` (novo):
```vba
Private Const BASE_DIR As String = "C:\Dev\projetos\Extratores"

Public Function PythonExe() As String
    PythonExe = BASE_DIR & "\venv\Scripts\python.exe"
End Function
Public Function ExtratorScript() As String
    ExtratorScript = BASE_DIR & "\src\extrator.py"
End Function
Public Function SetupSenhaScript() As String
    SetupSenhaScript = BASE_DIR & "\src\setup_senha.py"
End Function
Public Function SetupClienteScript() As String
    SetupClienteScript = BASE_DIR & "\src\setup_cliente.py"
End Function
```

Todos os módulos atualizados para usar `PythonExe()`, `ExtratorScript()`, `SetupSenhaScript()`, `SetupClienteScript()` em vez de `wsConfig.Range("B?").Value`.

**Para mudar de máquina ou pasta:** alterar apenas `BASE_DIR` em `ModConfig.bas`.
